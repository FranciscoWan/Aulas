# 🎨 Módulo de CSS

Bem-vindo ao repositório do **Módulo de CSS**!  
Aqui você encontrará os conteúdos, exemplos e exercícios que irão te ajudar a dominar os principais conceitos de estilização na web.

---

## 📚 Conteúdo das Aulas

### 📖 Aula 1 – Seletores e Box Model
- Tipos de seletores (elemento, classe, id, combinadores, pseudo-classes).
- Como funciona o **Box Model** (margin, border, padding, content).
- Diferença entre `inline`, `block` e `inline-block`.

---

### 📖 Aula 2 – Display, Position e Flex Direction
- Propriedade `display` (`block`, `inline`, `inline-block`, `flex`, `grid`).
- Tipos de `position` (`static`, `relative`, `absolute`, `fixed`, `sticky`).
- Flexbox: introdução ao `flex-direction`.

---

### 📖 Aula 3 – Grid Layout, Pseudo-elementos e Media Queries
- Fundamentos de **CSS Grid Layout**.
- Uso de **pseudo-elementos** (`::before`, `::after`).
- Responsividade com **media queries**.

---

### 📖 Aula 4 – Transições e Animações
- Criando efeitos suaves com **transitions**.
- Construindo animações com **@keyframes**.
- Boas práticas para animações performáticas.

---

### 🚀 Aula 5 – Projeto Final
**Desafio:**  
Desenvolver uma **página de portfólio pessoal** utilizando **HTML** e **CSS** (com possibilidade de adicionar **JavaScript** para interatividade).  

A página deve conter:
- 📌 Informações pessoais  
- 🛠️ Habilidades  
- 💼 Projetos realizados  
- ✉️ Formas de contato  

---

## 🖥️ Tecnologias Utilizadas
- **HTML5**
- **CSS3**
- (Opcional) **JavaScript**

---

## 🎯 Objetivo
Ao final deste módulo, você será capaz de:
- Criar layouts modernos e responsivos.
- Aplicar animações e interações em páginas web.
- Desenvolver um portfólio pessoal do zero.
